Video project

Via deze keuze krijgt u een overzicht te zien van alle documenten die bij dit project behoren.

 

Betekenis velden

Projectnummer

Dit is het projectnummer

Projecttype

Via dit veld kunt u het type project instellen. Afhankelijk van het type worden bepaalde instellen voor dat project in- of uitgeschakeld. Uit welke typen kunt u kiezen:

•Standaard project (veld leeg laten)

•Constructie-project aannemer (code 'CON' invullen)

Klantnummer

Dit is de klantnummer voor wie u die project maakt.

Architectnummer

Indien u als projecttype 'CON' invult, kunt u ook aangeven wie de bijbehorende architect is. Voor meer uitleg verwijzen we naar het aannemerbeheer.

Kortingcode

Voor ieder project kunt u een kortingcode toekennen.

Projectdatum

Dit is de datum waarop het project is toegevoegd.

Status

Via dit veld kunt u de status van het project instellen :

•A : Actief, het project is nog in uitvoering

•P : Gepauzeerd : het project is tijdelijk gestopt.

•E : Beëindigd: het project is beëindigd, voltooid.

Lev. Naam

Lev. Straat

Lev. Woonplaats

In deze velden kunt u de naam en adresgegevens van de klant invullen. Dit zijn niet altijd de gegevens van de ingevulde klantnummer (vb het klantnummer is een aannemer, terwijl de plaats van levering op de bouwwerf van zijn klant is).

 

 

Wanneer u een nieuw document (offerte, bon of factuur) toevoegt, kunt u een projectnummer invullen.

 

Bij alle verkoopdocumenten (offertes, bonnen, facturen) kunt u het projectnummer invullen. Door dit projectnummer in te vullen, wordt dat document automatisch gekoppeld aan dat project. Wanneer u nadien een overzicht opvraagt van alle document van dat project, wordt dit document getoont in de lijst.

 

 

Video project

Wanneer u de keuze <V> video project kiest, krijgt u een overzicht te zien van alle documenten die aan dit project zijn gekoppelt.

 

Keuzes in de AXITO-kader (video)

Toevoegen, Wijzigen, Schrappen, ...

Deze keuzes worden besproken in 'De verschillende keuzes'

Video document

Via deze keuze krijgt u het document direct te zien. Wanneer u bijvoorbeeld met de cursorbalk op een regel (record) staat dat een offerte is, dan kunt u via deze keuze <V>ideo document de video van deze offerte opvragen.

 

Betekenis velden

Document

Dit veld heeft het type document weer, zoals offerte, VK-bestelbon, VK-leveringsbon, Factuur, enz...

Document-nummer

Dit is het documentnummer van het betreffende document (zie vorig veld)

Lijstcode

Dit is de lijstcode voor dit record. Hiermee kunt u voor ieder document bepaalde codes ingeven, zoals bijvoorbeeld een status 'in uitvoering'.

De lijstcode '!' uitroepteken heeft een speciale functie. Wanneer u het uitroepteken invult, heeft dit tot gevolg dat dit document niet meer bestaat voor de berekening van totalen:

•Telt niet meer mee bij de berekening van het totaal aantal geleverde goederen).

•Wanneer u verschillende offertes voor een project heeft gemaakt, kunt u sommige offertes uitschakelen voor de berekening van het totaal aantal te leveren.

Omschrijving

Hierin kunt u een bijkomende omschrijving voor dit document invullen.

Datum

Dit is de datum van toevoeging aan het project.

Nota

Voor elke regel van dit project kunt u een bijkomende nota invullen.
