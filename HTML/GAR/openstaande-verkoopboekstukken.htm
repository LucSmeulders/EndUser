Openstaande verkoopboekstukken

 

 

Via deze keuze kunt u lijsten opvragen van alle klanten en hun facturen waarvoor ze nog een betaling moeten uitvoeren. Ook deze lijsten spreken voor zichzelf. Afhankelijk van welke klanten en openstaande facturen u wilt zien, heeft u de keuze tussen:
 

•Openstaand, reeds vervallen

•Openstaand, nog te vervallen

•Openstaand

•Te betalen creditnota's

 

Vervolgens verschijnen op het scherm de verschillende openstaande verkoopboekstukken.

 

 

Mogelijke keuzes

 

Printen aanmaningen

Via de keuze <P>rinten kunt u voor ieder openstaand verkoopboekstuk een aanmaning (rappel) afdrukken. Hierbij kunt u kiezen uit 3 verschillende modelbrieven, afhankelijk van de reeds vervallen termijn.

 

U kunt hiervoor kiezen uit volgende mogelijkheden:

 

Klant xxxxxx volledig: Computer bepaalt type aanmaning

Hierbij wordt voor klant xxxxxx één aanmaning geprint voor alle openstaande verkoopboekstukken. De computer bepaald hierbij welk type aanmaning zal worden geprint.

 

Klant xxxxxx volledig: Zelf bepalen type aanmaning

Zie vorige. Nu bepaalt u zelf welk type aanmaning wordt geprint.

 

Klant xxxxxx gedeeltelijk: Computer bepaalt type aanmaning

Hierbij kunt u met behulp van <+> en < - > bepalen welke verkoopboekstukken op de aanmaning voor klant xxxxxx  worden opgenomen. De computer bepaald hierbij welk type aanmaning zal worden geprint.

 

Klant xxxxxx gedeeltelijk: Zelf bepalen type aanmaning

Zie vorige. Nu bepaalt u zelf welk type aanmaning wordt geprint.

 

Alle klanten volledig: Computer bepaalt type aanmaning

Hierbij wordt voor alle klanten één aanmaning geprint voor alle openstaande verkoopboekstukken. De computer bepaald hierbij welk type aanmaning zal worden geprint.

 

Alle klanten volledig: Zelf bepalen type aanmaning

Zie vorige. Nu bepaalt u zelf welk type aanmaning wordt geprint.

 

Alle klanten gedeeltelijk: Computer bepaalt type aanmaning

Hierbij kunt u bepalen welke verkoopboekstukken op de aanmaning van alle klanten worden opgenomen. De computer bepaalt hierbij welk type aanmaning zal worden geprint.

 

Alle klanten gedeeltelijk : Zelf bepalen type aanmaning

Zie vorige. Nu bepaalt u zelf welk type aanmaning wordt geprint.

 

Boekstuk 9xxxxxx

Hierbij kunt u een aanmaning printen, die betrekking heeft op dit boekstuk.

 

Wijzigen instellingen aanmaning

Via deze keuze kunt u de teksten enz. van de 3 aanmaningen aanpassen.

@E-mail

Deze keuze kunt u gebruiken wanneer u uw aanmaningen rechtstreeks wilt e-mailen naar de klant.

Bank-boeking

Via deze keuze kunt u het openstaande boekstuk betalen m.b.v. een bankboeking. Wanneer u hierna het betalingsprogramma van de bank start, kunt u deze geregistreerde betaling op een eenvoudige manier uitvoeren.

# Wijzigen vervaldata

Via deze keuze kunt u de vervaldata wijzigen.
