Colli - Leeggoed

Via deze module kunt u voor een artikel aangeven hoe deze kunnen verkocht worden (in een doos, kist, krat). Algemeen worden dit collies genoemd. Een colli is een herbruikbare verpakking met een eigen waarde. Deze colli kan worden teruggenomen, waardoor de waarde van die colli ook wordt terugbetaald.

 

Opmerking:

 

Enkel via de 'Groenten- en fruitversie' kunt u deze module gebruiken.

 

Instellen van een colli-artikel

Voor het instellen van een colli-artikel kunt u kiezen uit 2 werkwijzes. Afhankelijk van uw situatie kiest u de geschikt werkwijze.

 

Basis colli-artikel

Als voorbeeld nemen we bananen: deze worden geleverd in een colli en in 1 colli gaan 6 bananentrossen. Een bananentros wordt per stuk verkocht (met een prijs per bananentros), en de colli waarin deze geleverd worden heeft ook een waarde. Uiteraard wordt deze colli bij een volgende levering mee teruggenomen.

Voor een bananentros kunt u de collie-gegevens instellen. In het macro-scherm ziet u rechts een veld 'Code colli'. Wanneer u dit veld wijzigt, krijgt u een dialoogvenster:

 

Code werkwijze

Door hierin een 'B' in te vullen selecteert u de 'Basis werkwijze'.

Code colli

Via dit veld kunt u ofwel een vast aantal ingeven (vb. 6) of een willekeurige code (vb. B). Bij een willekeurige code zal bij de verkoop opgevraagd worden hoeveel producten er in de colli zitten. In veel gevallen is immers het aantal op voorhand niet vastgelegd en moet dit op het moment zelf ingevuld worden.

Colli inhoud

Hoeveel artikelen (bananentrossen) gaan er in 1 colli.

Collie eenheidsprijs

Dit is de waarde van 1 colli.

Verkoopprijs incl.btw - Verkoopprijs excl.btw

Dit is de prijs voor 1 bananentros.

 

Uitgebreid colli-artikel

Als voorbeeld nemen we CocaCola. Wanneer je een bak CocaCola koopt, koop je de drank. Maar daarbij ook het flesje waar de drank in zit en de bak waar de flesjes inzitten. Daarnaast kun je een bak van 6, 12 of 24 flesjes kopen.

In het macro-scherm ziet u rechts een veld 'Code colli'. Wanneer u dit veld wijzigt, krijgt u een dialoogvenster:

 

 

Code werkwijze

Door hierin een 'U' in te vullen selecteert u de 'Uitgebreide werkwijze'.

Code colli

Via dit veld kunt u ofwel een vast aantal ingeven (vb. 6) of een willekeurige code (vb. B). Bij een willekeurige code zal bij de verkoop opgevraagd worden hoeveel producten er in de colli zitten. In veel gevallen is immers het aantal op voorhand niet vastgelegd en moet dit op het moment zelf ingevuld worden.

Leeggoed artikelnummer

Dit is het colli-artikel waarin het product wordt verpakt. In ons voorbeeld is dit het flesje.

Colli inhoud

We hebben de mogelijkheid voorzien om tot 5 verschillende colli inhouden te beheren.

Aantal

Hoeveel artikelen zijn er in een colli.

Prijs art. ex.BTW - Prijs art. in. BTW

Wat is de verkoopprijs van dit colli.

Colli artikelnummer

Dit is het artikelnummer van het colli. In ons voorbeeld is dit de bak waarin de flesje CocaCola zitten.

Verkoopprijs incl.btw

Verkoopprijs excl.btw

Dit is de prijs voor 1 cola zonder colli (= zonder het flesje en zonder de bak).

 

Leeggoed artikel

Uiteraard moet u ook de colli-artikelen (of leeggoed) toevoegen. Een colli-artikel is een gewoon artikel, echter het heeft GEEN verkoopprijs. Wel moet u de colli-gegevens correct invullen:

 

Code werkwijze

Door hierin een 'B' in te vullen selecteert u de 'Basis werkwijze'.

Code colli

Hier vult u 1 in.

Colli inhoud

Hier vult u 1 in.

Collie EH-prijs

Dit is de waarde van 1 colli. In ons voorbeeld is dit de prijs van 1 flesje.

Verk.pr.BTW - Verkoopprijs

Hier vult u 0 in.
