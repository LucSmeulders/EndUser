AXITO en Windows

Windows 7, 8 en windows 10 vormen totaal geen probleem voor AXITO!

 

Een klein beetje extra uitleg:

•De AXITO windows-versie draait sowieso onder eender welke windows-versie (dus ook bijv. Windows 7 64-bits)

•De AXITO hybride-versie (klassieke versie) draait wel onder de 32-bits Windows 7, maar NIET onder de 64-bits Windows 7.

 

 

Opmerking:

Als u als oudere AXITO-klant per se de AXITO hybride-versie wilt blijven draaien, neem dan de 32-bits versie van Windows 7.
